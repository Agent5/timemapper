HyperNotes Information Organization for the HyperText Age

This is the project home page for the HyperNotes project. The project is a rewrite and updating of the original Microfacts codebase (2007-2011) developed for the Weaving History project. The rewrite is in pure javascript (NodeJS and Backbone).

Work is still at a relatively early stage so for documentation including install information please see the HyperNotes GitHub repo.
