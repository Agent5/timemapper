FAQ

What Structure must the Spreadsheet Have?

TimeMapper recognizes certain specific columns / fields in your spreadsheet data. Best overview is the template. Note that not all fields are required (strictly, the only required fields are title and start field, though even start can be omitted if you only want a map).

Note that you can also add any number of other columns beyond those that TimeMapper uses.

Date Formats

Preferred date format is ISO 8601 (e.g. 2000-01-01), but TimeMapper recognizes most types of date.
